## 🎯 什么是 PolyBot？

PolyBot 是一个用于 [Polymarket](https://polymarket.com)（全球最大的预测市场平台）的**全自动交易机器人**。

### 主要特性

| 特性 | 描述 |
|------|------|
| 🤖 **完全自动化** | 24/7 全天候交易 |
| 🌐 **Ferox 控制面板** | 通过漂亮的网页界面管理机器人 |
| 🛡️ **风险管理** | 熔断器保护和损失限制 |
| 📊 **信号引擎** | 5个技术指标协同工作 |
| 💰 **Solana 资金** | 自动 SOL→USDC 桥接 |
| 🧪 **模拟模式** | 无需真金白银测试 |

## 🚂 在 Railway 上部署

### 第1步：创建账户

1. 访问 [railway.app](https://railway.app)
2. 点击 "Login"
3. 使用 "Login with GitHub"（推荐）

### 第2步：创建新项目

1. 点击 **"New Project"**
2. 选择 **"Deploy from GitHub repo"**
3. 搜索并选择 **"PolyBot"**

### 第3步：配置环境变量

进入 **Variables** 并添加：

| 变量 | 值 | 必需 |
|------|-----|:----:|
| `DRY_RUN` | `true`（用于测试） | 推荐 |
| `MODE` | `signal`、`copy`、`arbitrage` 或 `all` | 可选 |
| `POLYGON_PRIVATE_KEY` | 你的钱包私钥 | 真实交易必需 |

### 第4步：验证部署

1. 等待部署变成 **绿色** ✅
2. 打开 Ferox 控制面板
3. 查看机器人状态
4. 如果显示在线，就成功了！🎉

## 🎯 交易模式

### 1. 信号交易 (Signal)

信号引擎使用5个技术指标分析市场：

- **移动平均线交叉 (MA)** - 27%
- **相对强弱指数 (RSI)** - 27%
- **MACD** - 23%
- **动量** - 13%
- **成交量** - 10%

```bash
# 激活信号交易：
MODE=signal
```

### 2. 跟单交易 (Copy)

复制成功交易者的操作：

```bash
# 激活跟单交易：
MODE=copy
COPY_WALLETS=wallet1,wallet2,wallet3
```

### 3. 套利 (Arbitrage)

寻找 YES + NO < $1.00 的机会：

```bash
# 激活套利：
MODE=arbitrage
```

### 4. 所有模式

```bash
# 使用所有模式：
MODE=all
```

## 🌐 Solana 集成

PolyBot 可以从 Solana 自动接收资金。

### 配置变量

| 变量 | 描述 | 默认值 |
|------|------|--------|
| `SOLANA_PRIVATE_KEY` | Solana 私钥 | （必需） |
| `AUTO_SWAP_SOL` | 自动 SOL→USDC 兑换 | `true` |
| `SOL_SWAP_THRESHOLD` | 最小兑换 SOL | `0.1` |
| `MIN_POLY_BALANCE_USDC` | 自动资金的最低余额 | `20.0` |
| `BRIDGE_FUND_AMOUNT` | 最大转账金额 | `50.0` |

### 资金流程

1. 向机器人地址发送 SOL
2. 机器人检测到 SOL
3. 通过 Jupiter DEX 兑换 SOL → USDC
4. 通过 Wormhole 转账 USDC 到 Polygon
5. 准备好交易！

## 🛡️ 风险管理

PolyBot 包含多层保护：

```
┌────────────────────────────────────────────────────────────────┐
│                    🛡️ 风险管理系统                              │
├────────────────────────────────────────────────────────────────┤
│                                                                │
```

| 保护 | 描述 |
|------|------|
| **每日损失限制** | 达到 -5% 每日损失时停止交易 |

| **凯利仓位** | 根据优势计算最优仓位大小 |
| **最大回撤** | 可配置的回撤限制 |
| **冷却期** | 触发保护后的暂停期 |

## 🔧 故障排除

### 常见问题

| 问题 | 解决方案 |
|------|----------|
| 机器人不响应 | 检查 Railway 日志是否有错误 |
| 交易未执行 | 验证 `DRY_RUN=false` 并设置了钱包密钥 |
| 余额显示 0 | 确保你的 Polygon 钱包有 USDC |
| API 错误 | 检查环境变量是否正确设置 |

### 检查日志

在 Railway 中：
1. 进入你的项目
2. 点击 "Deployments"
3. 选择最新的部署
4. 点击 "View Logs"

### 常见错误信息

| 错误 | 含义 | 解决方案 |
|------|------|----------|
| `Insufficient balance` | USDC 不够 | 添加更多资金到钱包 |
| `Rate limited` | API 请求太多 | 等待几分钟 |
| `Circuit breaker active` | 连续亏损触发保护 | 等待冷却期或重置 |
